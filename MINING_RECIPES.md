## Цепочки крафта

### Железная цепочка
```
Железная Руда -> Железо -> Железный Слиток
```

### Медная цепочка
```
Медная Руда -> Медь -> Медный Слиток
```

### Бронзовая цепочка
```
Железная Руда + Медь -> Бронза -> Бронзовый Слиток
```

### Торовая цепочка
```
Железная Руда + Железо -> Тор -> Слиток Тора
```

### Платиновая цепочка
```
Железная Руда -> Платина -> Платиновый Слиток
```

### Сумеречные материалы
```
Железный Слиток + Тор + Платина -> Сумеречная Сталь
Железный Слиток + Слиток Тора + Платина + Сапфиры + Рубины -> Сумеречный Антрацит
```
